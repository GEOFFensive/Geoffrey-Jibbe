Les 3: Uitleg HTML, CSS en opgaves

Tijdens les 3 hebben we kennis gemaakt met het opstellen van een website.
Dit hebben we gedaan aan de hand van 'HTML-Kit' gecombineerd met een naar eigen keuze 'css-tool'.
De css-tool die wij gekozen hebben om onze lay-out in te verwerken is kladblok.


HTML

HTML staat voor HyperText Markup Language, en is dus een eigen programmeertaal vergelijkbaar met
bijvoorbeeld Java, Python en dergelijke.
Deze taal is speciaal ontworpen voor websites en wordt dus ook meestal voor dergelijke toepassingen gebruikt.
HTML-documenten bevatten zowel een tekstuele inhoud als een opmaak.
Om de opmaak later nog makkelijker te kunnen veranderen, wordt aangeraden via een CSS-tool te werken.
Deze CSS-tool kan gelinkt worden aan het HTML-bestand.
Hierdoor moet niet meer in het originele HTML-bestand gewerkt worden om de opmaak van de site te veranderen.
Dit zorgt ervoor dat men niet het gehele HTML-bestand moet overlopen en dat er geen risico meer is op het
veranderen van de tekstuele inhoud.


CSS

De CSS-tool, in ons geval kladblok, bevat de code voor de gehele opmaak van de site.
Hierin wordt "per categorie van tekst" een andere code voorzien om de opmaak tussen
de verschillende soorten tekst te onderscheiden.
Alle titels hebben bijvoorbeeld éénzelfde opmaak, net zoals alle tussentitels, tekst, etc.
Uit ondervinding is kladblok een CCS-tool waarbij niets van code is voorgeprogrammeerd,
maar hierdoor leert men wel de code interpreteren.


Opgaves

Vooraleer te beginnen met de opgaves, hebben we een presentie bijgewoond over HTML en CSS.
Hierna gingen we aan de slag met 4 opgaves.
Opgave 1 bestond eruit 5 HTML-bestanden aan te maken, om hier later mee verder te werken.
Deze 5 bestanden gaven de algemene structuur weer van een standaard website.
Opgave 2 hield in dat we in de voorgaande HTML-bestanden enkele wijzigingen aanbrachten, die voor onze website van toepassing waren.
In opgave 3 werd een CSS-bestand aangemaakt met standaard commando's voor het veranderen van de kleur van een stuk tekst.
Opgave 4 was een uitbreiding van opgave 3, hierin kwamen al iets gevorderdere commando's aan bod voor het veranderen van de algemene opmaak van een tekst.
